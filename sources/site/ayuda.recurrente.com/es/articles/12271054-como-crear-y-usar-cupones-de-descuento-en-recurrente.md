# ¿Cómo crear y usar cupones de descuento en Recurrente?

Aprende a crear códigos de descuento y descuentos por banco emisor para ofrecer promociones en tu tienda de Recurrente.

Con los cupones de descuento puedes ofrecer promociones especiales a tus clientes en la tienda de Recurrente. Puedes crear códigos de descuento que tus clientes ingresan al pagar.

Este artículo muestra el flujo desde la app móvil. Si usas la versión web, los pasos son los mismos pero la interfaz puede verse un poco diferente.

# Paso a paso

**Paso 1:** Ve a **Tienda** en el menú principal.

**Paso 2:** Haz clic en **Descuentos**.

[![](https://recurrente.intercom-attachments-1.com/i/o/q0how4fq/2327121522/0e4a8145d8f737f6a5abee81db67/8tggi4.png?expires=1784052900&signature=839d6657aee560c70bd910c45b4133abaf3d9f98764c3ddd9383b17e34febaa9&req=diMlEch8nIRdW%2FMW1HO4zfb9h4dW7U2Rs%2Bu166diT8rfJRACHUStBbiXmv%2Bt%0AfRMnI2t1Qslesrih%2Bcg%3D%0A)](https://recurrente.intercom-attachments-1.com/i/o/q0how4fq/2327121522/0e4a8145d8f737f6a5abee81db67/8tggi4.png?expires=1784052900&signature=839d6657aee560c70bd910c45b4133abaf3d9f98764c3ddd9383b17e34febaa9&req=diMlEch8nIRdW%2FMW1HO4zfb9h4dW7U2Rs%2Bu166diT8rfJRACHUStBbiXmv%2Bt%0AfRMnI2t1Qslesrih%2Bcg%3D%0A)

**Paso 3:** Haz clic en **Añadir** para crear un nuevo descuento.

**Paso 4:** En **Tipo de descuento**, selecciona **Código**.

**Paso 5:** En **Código de descuento**, escribe el código que tus clientes usarán (por ejemplo: PRIMAVERA20). El código se convierte automáticamente a mayúsculas.

**Paso 6:** En **Valor del descuento**, elige el tipo:

- **Porcentaje:** descuenta un porcentaje del total (ej. 20%)

- **Monto fijo:** descuenta una cantidad fija (ej. Q25.00)

**Paso 7:** En **En caso de una suscripción, aplica:**, elige la duración:

- **En el primer pago:** el descuento solo aplica al primer cobro de una suscripción

- **En todos los pagos:** el descuento aplica a todos los cobros recurrentes

**Paso 8:** Activa o desactiva la opción **¿Está activo?**

**Paso 9:** Haz clic en **Añadir** para guardar. Tu cupón aparecerá en el listado de descuentos.

[![](https://recurrente.intercom-attachments-1.com/i/o/q0how4fq/2327121570/b4e58b48d15c82fc8be125a67614/977hu7.png?expires=1784052900&signature=9e0a2d937c460575b5d679a1ebdd25381cc28da280e2c4584eba5f808d091d4f&req=diMlEch8nIRYWfMW1HO4zfvJU4bea%2Bwha72IPc5NA5ucvHpxGCGtZF125vEK%0ApFTLQnHO0zqFNQCkvDY%3D%0A)](https://recurrente.intercom-attachments-1.com/i/o/q0how4fq/2327121570/b4e58b48d15c82fc8be125a67614/977hu7.png?expires=1784052900&signature=9e0a2d937c460575b5d679a1ebdd25381cc28da280e2c4584eba5f808d091d4f&req=diMlEch8nIRYWfMW1HO4zfvJU4bea%2Bwha72IPc5NA5ucvHpxGCGtZF125vEK%0ApFTLQnHO0zqFNQCkvDY%3D%0A)

**Paso 10:** Si después necesitas hacer cambios, haz clic en el cupón para editarlo, o desactívalo para que los clientes no puedan usarlo temporalmente.

# Descuento por banco emisor

También puedes crear descuentos que se aplican automáticamente cuando el cliente paga con una tarjeta de un banco específico. Para crear un descuento por banco emisor, sigue los mismos pasos pero selecciona **Banco emisor** en **Tipo de descuento** y elige los bancos que aplican.

👉 Para más detalles, consulta [Cómo crear un descuento por banco emisor](https://ayuda.recurrente.com/es/articles/14011483-como-crear-un-descuento-por-banco-emisor-en-recurrente).

# Cómo aplicar un cupón en una compra

Si creaste un cupón de código, tus clientes pueden usarlo así:

**Paso 1:** El cliente abre el link de pago o tu tienda.

**Paso 2:** En la pantalla de checkout, haz clic en **Ver detalles** para expandir el resumen del pago.

**Paso 3:** Selecciona el campo de **código de descuento** e ingresa el código.

**Paso 4:** El descuento se aplica y se muestra con el porcentaje y el monto descontado.

![](https://recurrente.intercom-attachments-1.com/i/o/q0how4fq/2327121591/6c7d7ef116f1576d8b01e8060f43/2p10q0.png?expires=1784160000&signature=146123ed2f96ea82b949912424c05dc2b1c1e8067da444b2c852908b9950cfdb&req=diMlEch8nIRWWPMW3Hu4gZLWBw8ESKPsBfzVcVU0AseQ2pnLH3QQkQs%2FzEn8%0ASQ%3D%3D%0A) ![](https://downloads.intercomcdn.com/i/o/q0how4fq/2330171338/8505f7f8dda5fce5867cd721225b/rbsbvd.png?expires=1784160000&signature=39e2605869331cc28fb3d264753e77156a120217fa279f6cd032bc23e0947283&req=diMkFsh5nIJcUfMW3Hu4gfzBNVZjt5uVXiE2zKet60lmHFPfGlm0W6XXYQns%0AKw%3D%3D%0A)

## Preguntas frecuentes

**¿Se pueden usar varios cupones a la vez?**

No. Solo se puede aplicar un descuento por compra. Si el cliente ingresa un código nuevo, este reemplaza el anterior.

**¿Qué pasa si mi cliente tiene un código y también paga con un banco con descuento?**

El código de descuento tiene prioridad. Si el cliente ya aplicó un código, el descuento por banco no se activa.

**¿Puedo desactivar un cupón sin eliminarlo?**

Sí. Selecciona el cupón y desactiva el interruptor **¿Está activo?**. Los clientes no podrán usarlo mientras esté inactivo, pero puedes reactivarlo en cualquier momento.

**¿El descuento aplica para pagos únicos y suscripciones?**

Sí. Para suscripciones, tú decides si el descuento aplica solo al primer pago o a todos los cobros recurrentes al momento de crear el cupón.

**¿Puedo usar el asistente de IA para gestionar cupones?**

Sí. Si tu cuenta tiene habilitado el asistente de IA, puedes crear, modificar, eliminar y consultar tus cupones directamente desde el chat. Solo escribe tu solicitud en lenguaje natural, por ejemplo: "Crea un cupón VERANO30 con 30% de descuento".
