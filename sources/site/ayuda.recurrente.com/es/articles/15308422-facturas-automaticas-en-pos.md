Conoce cómo se emite la factura después de un cobro en POS, con NIT o como Consumidor Final.

En POS, la factura automática se emite después de que el pago fue aprobado. Primero cobras; luego puedes ingresar el NIT del cliente. Si dejas el NIT vacío, la factura se emite como CF / Consumidor Final.

​

# Antes de empezar

- Este artículo explica cómo funciona la factura durante un cobro en POS.

- No cubre cómo activar o configurar la facturación automática.

- El campo de NIT aparece después de completar el pago, no antes.

​

# Cómo funciona el flujo

1. Crea la orden o inicia el cobro desde POS.

2. Completa el pago con el método disponible en tu POS.

3. Cuando el pago sea aprobado, verás la pantalla de confirmación.

4. En esa pantalla puedes ingresar el **NIT para la factura**. Es opcional.

5. Presiona **Continuar**.

6. Recurrente emite la factura automáticamente si tu cuenta tiene facturación activa.

7. Después puedes enviar el recibo, imprimir el comprobante o imprimir la factura desde el POS.

​

# Si el cliente te da NIT

Ingresa el NIT en la pantalla que aparece después del pago aprobado.

Al presionar **Continuar**, Recurrente usa ese NIT para emitir la factura electrónica y asociarla al cobro.

​

# Si no ingresas NIT

Si dejas el campo vacío y presionas **Continuar**, Recurrente emite la factura como **CF / Consumidor Final**.

Este flujo sirve cuando el cliente no solicita factura con datos específicos.

​

# Enviar la factura por correo

Después de emitir la factura, POS te muestra la pantalla para enviar el recibo.

Si ingresas el correo electrónico del cliente, Recurrente envía el recibo del pago. Cuando la factura ya fue emitida, el correo incluye el enlace para descargarla.

​

# Imprimir comprobante o factura

Desde la pantalla final del POS puedes imprimir:

- El comprobante del pago.

- La factura, si ya fue emitida correctamente.

​

# Dónde descargarla después

También puedes encontrar la factura desde **Actividad**.

Abre el cobro correspondiente y busca la sección de **Factura electrónica** para descargarla. Si estás usando el POS, también puedes reimprimirla desde las acciones del cobro cuando la factura esté disponible.

​

Revisa el NIT antes de continuar. Si la factura se emite con datos incorrectos, puede que necesites anularla o emitir una nueva según corresponda.
