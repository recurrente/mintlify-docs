# ¿Cómo puedo solicitar una ampliación para un link de pago en Recurrente Personal?

4 de abril de 2024

Por el momento, esta opción solo está disponible para cuentas empresariales y no para cuentas personales.

Si deseas saber más puedes contactarnos a [soporte@recurrente.com](mailto:soporte@recurrente.com)
